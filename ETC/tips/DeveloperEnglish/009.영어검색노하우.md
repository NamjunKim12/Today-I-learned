### 특정 웹사이트에서만 검색 결과 가져오기

- 'filetype: 확장자' : 특정 확장자를 가진 파일만 검색
> - Python Cheat Sheet filetype:pdf
- 검색 키워드의 *는 와일드카드(wildcard) 역할
> - "What a *day" 검색=> "What a nice day", 'What a bad day', 'What a good day' 등 모두 검색
- 마이너스(-) 연사자는 특정 키워드를 제외한 검색 결과를 보여준다.
> - Git hosting solution -github +gitlab 검색 => Git 호스팅 솔루션 검색결과에서 github라는 단어를 포함하지 않고, gitlab이라는 단어를 포함한 결과를 보여준다.

### 소프트웨어에 특화된 영단어로, 간결하게

- 구글은 `단순한 검색 키워드`부터 시작해 키워드를 추가해나가는 것을 권장. 지나치게 긴 검색 키워드는 검색 정확도를 떨어뜨림
- 장황한 문장이 아닌 핵심 소프트웨어 특화 영단어로 간결하게 작성


### 웹페이지가 사용할만한 키워드로 검색하기

- 검색키워드는 회화 키워드가 아닌 소프트웨어에서 자주 사용하는 키워드로 작성
> - solving memory leaks in java => `troubleshooting` memory leaks in java
> what are the good ways when configuring an nginx server => `best practices` for nginx configuration
